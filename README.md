# 👾 Space Invaders Web | Space Invaders Web Game

## 📌 Sobre | About

**PT-BR:**  
Este é meu **primeiro grande projeto em desenvolvimento web**, inspirado no clássico jogo de arcade dos anos 70 **Space Invaders**.
O projeto foi desenvolvido utilizando **HTML**, **CSS** e **JavaScript**, com o objetivo de praticar lógica de programação, manipulação do DOM e estruturação de um jogo simples no navegador.

Como diferenciais do projeto, implementei:

- **Inserção de imagens** para representar os elementos do jogo  
- **Efeitos sonoros** para tornar a experiência mais imersiva  
- **Sistema de hitbox** para detecção de colisões entre tiros e inimigos  

<br>

**EN:**  
This is my **first large web development project**, inspired by the classic 1970s arcade game **Space Invaders**.
The project was built using **HTML**, **CSS**, and **JavaScript**, aiming to practice programming logic, DOM manipulation, and structuring a simple browser game.

New elements implemented in this project include:

- **Image assets** to represent game elements  
- **Sound effects** to improve immersion  
- **Hitbox system** for collision detection between bullets and enemies
